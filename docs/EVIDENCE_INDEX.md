# Evidence Index

This document is intended to be generated automatically by `scripts/evidence_index.ps1` and used as a presentation aid during submission and viva.

## What it captures

1. Latest reports and their sizes.
2. Submission bundle path.
3. A recommended evidence sequence for judging and viva.

## Why it helps

1. It turns scattered artifacts into one clear audit trail.
2. It helps you point examiners to the most relevant outputs immediately.
3. It supports a repeatable demo path instead of ad hoc manual browsing.
